### Descrição

O projeto consome uma [API](https://pokeapi.co/) onde obtém os dados dos Pokémons.
O projeto usa vite, vue 3, pinia e vue router.

Para executar o projeto rode os seguintes comandos:

`npm install`
`npm run dev`
